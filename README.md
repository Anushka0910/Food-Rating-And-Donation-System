# Food Rating and Donation System

## 📌 Project Overview
The Food Rating and Donation System is a web-based platform designed to help users rate food dishes from hotels and reduce food waste by enabling hotels or restaurants to donate leftover food.

The system allows customers to rate dishes using a star rating system and view ratings for different hotels. Additionally, hotels can post leftover food available for donation so that users can access and benefit from it.

This platform aims to improve food quality transparency and contribute to reducing food waste.

---

## 🚀 Features
- User Signup and Login
- Dish Rating System (5-Star Rating)
- View Dish Ratings by Hotel
- Hotels can post leftover food for donation
- Users can view available donated food
- QR Code and Location-based verification for rating dishes
- Simple and user-friendly interface
